# Limón Method V2

A two-phase AI-powered exam preparation pipeline for any subject, any exam board.

---

## What it does

Most exam prep tools make you do the heavy lifting — searching for notes, guessing what's important, hoping your practice questions are relevant. Limón flips that.

You drop your past papers. Claude does the rest.

---

## How it works

### Phase 1 — Immediate answers
The moment you paste or upload question papers, Claude reads every question and answers all of them. No prompting required. You get a complete, labeled set of answers before you've even asked for anything.

### Phase 2 — Say "Limón"
The codeword unlocks the full pipeline:

1. **Pattern analysis** — Claude maps which topics appear most, how marks are distributed, which command words examiners love, and which topics rotate vs. always show up.

2. **Structured notes** — Detailed, plain-language notes built for both human reading and NotebookLM processing. Every topic follows the same structure: Key Concepts, How It Works, Common Exam Angles, Key Terms, and Watch Out For.

3. **4 practice sets** — Generated using the school repetition model. Same concepts, same difficulty, increasing surface variation across sets 1 through 4. Each set comes with its own separate answer key. No shuffling. No arbitrary difficulty spikes. Just the kind of repetition that actually builds exam confidence.

---

## Why it works

Schools have been repeating questions for decades because repetition with variation is how retention actually happens. Limón automates that logic. By set 4, you've seen every core concept from enough angles that no real exam question should catch you off guard.

---

## Compatibility

- Any subject — sciences, humanities, mathematics, languages, social studies
- Any exam board
- Notes are formatted for direct import into NotebookLM
- Works with Claude on claude.ai as an installable skill
